# Weather Forecast Application 🌦️

A modern and responsive weather forecast application that provides real-time weather updates and a 7-day weather forecast for locations worldwide. Users can search for weather information by city name or use their current location to instantly view accurate weather conditions.

## 🚀 Features

* 🔍 Search weather by city name
* 📍 Detect and use current location
* 🌡️ Real-time temperature display
* ☁️ Weather condition information
* 💧 Humidity details
* 🌬️ Wind speed information
* 📅 7-Day weather forecast
* 📱 Fully responsive design
* ⚡ Fast and dynamic weather updates


## 🎯 How It Works

1. Enter a city name in the search box.
2. Click the **Search** button.
3. The application fetches weather data from the API.
4. Current weather details are displayed.
5. A 7-day forecast is shown below the current weather card.
6. Users can also click **Use My Location** to fetch weather data for their current location.
